# hatenablog-review

[はてなブログ](https://hatenablog.com/)のブログテーマ「Review」です。

テーマストアから[インストールして使う](http://blog.hatena.ne.jp/-/store/theme/8454420450101088484)ことができます。

## 開発

セットアップ：

```
yarn install
```

Sassの監視とコンパイルを開始：

```
gulp
```

はてなブログの「デザインCSS」に設定する：

```
@import 'http://localhost:3030/app.css';
```
